This Python application performs curve fitting using algebraic and trigonometric polynomials, and also includes basic DCDFT analysis.

Inspired by I.L.Andronov's MCV [1994OAP.....7...49A, 2004AstSR...5..264A]

Requirements:
Python 3.8+
* scipy
* matplotlib
* pandas

Tested on Windows 7, 10, 11; Debian 13; Linux Mint
